# Transformation & Post-Processing

Files and scripts found here are primarily for the purpose of post-processing data tranformation when
using the carrot-transform tool (https://carrot.ac.uk/transform).
The scripts are based on the analysts own experiences when transforming their own data within their
Trusted Research Environments (TRE).

Scripts include
- Production of OMOP CDM domain tables which are currently unsupported by carrot-transform.
- Fixes for anomalies from limitations with the carrot-transform output pending any future fixes to
  the software.
- Quality assurance scripts
